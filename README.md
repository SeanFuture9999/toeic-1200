# TOEIC 1200 - 30 天多益單字衝刺班

30 天快速學習 1200 個多益高頻單字的互動式網頁應用程式。

## 功能

- **30 天排程**：每天 40 字，6 種學習法輪替
- **翻卡學習**：點擊翻面，附 KK 音標 + 記憶法
- **聽寫模式**：聽發音選正確單字（Web Speech API）
- **混淆對決**：易混淆字對照比較 + 秒殺口訣
- **字根拆解**：30 組字根，每組 5 字
- **詞族家族樹**：一個字根學完所有詞性
- **職場黑話**：外商常用表達解碼
- **進度追蹤**：localStorage 記錄學習進度與連續天數

## 學習方法

| 方法 | 說明 | 單字數 |
|------|------|--------|
| 短字優先 | 字母少、意思多的高 CP 值字 | 70 |
| 混淆對決 | 長相像但意思不同的陷阱字 | 50 對 |
| 字根拆解 | 同字根衍生字群 | 150 |
| 詞族家族樹 | 同根不同詞性 | 120 |
| 地雷 + 諧音 | 易記錯字 + 聯想記憶 | 80 |
| 職場黑話 | 外商 Slack 常見表達 | 30 |

## 技術

- 純 HTML / CSS / JavaScript（無框架）
- ES Modules
- Web Speech API 語音朗讀
- localStorage 進度持久化
- Mobile-first 響應式設計
